# Relative

하위 컴포넌트들이 left, top, bottom, right 프로퍼티를 사용해 자신의 원래 정해진 위치에서 상대적으로 변위를 시키고 싶을 때
사용하는 부모 컴포넌트. 주로 `<Absolute>`와 함께 사용됩니다.

## Dependencies
<App>
<Flex>
<Chip bg="blue.deep" color="#FFF">none</Chip>
</Flex>
</App>

## Properties

|Property|Type|Default|
|-|-|-|
| top | ResponsiveValue | 0 |
| bottom | ResponsiveValue | 0 |
| left | ResponsiveValue | 0 |
| right | ResponsiveValue | 0 |
| z-index | ResponsiveValue | 0 |

    </Relative>
